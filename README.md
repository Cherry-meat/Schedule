# StudentPlanner  
Система управления персональным расписанием студента.

## О проекте

### Почему был создан этот проект?
Проект создан для помощи студентам в организации учебного процесса.

### Какую проблему он решает?
Приложение помогает студентам отслеживать учебные события, управлять задачами и планировать своё время, предотвращая пропуск важных занятий и дедлайнов.

### Ключевые преимущества:
- **Приватность и безопасность:** логины и пароли защищены хешированием
- **Быстродействие:** лёгкий код обеспечивает высокую скорость работы
- **Доступность:** веб-интерфейс позволяет пользоваться системой с любого устройства

## Быстрая установка и запуск

### Установите необходимое ПО:
1. Установите Go 1.21 по ссылке выше
2. Установите Node.js для запуска клиентской части
3. Установите PostgreSQL. Во время установки запишите заданные вами данные для подключения (логин, пароль, порт)

### Настройка базы данных:
1. Откройте командную строку PostgreSQL (или общую командную строку/терминал)
2. Создайте базу данных:
   ```bash
   createdb -h localhost -U postgres student_planner
   ```
3. Запустите приложение - таблицы создадутся автоматически при первом запуске

### Запустите приложение:
1. Скачайте исходный код проекта
2. Запустите сервер из командной строки в корневой директории проекта:
   ```bash
   go run .
   ```
3. В отдельной командной строке перейдите в директорию клиента и запустите:
   ```bash
   npm install // только при первом запуске
   npm start
   ```

## Возможности
- **Учёт событий:** добавление, редактирование и удаление учебных событий
- **Управление задачами:** создание, отслеживание и выполнение учебных задач
- **Ключевые даты:** фиксация даты и времени занятий
- **Полная изоляция данных:** ваше расписание и задачи доступны только вам
- **Календарь событий:** визуализация расписания на день и неделю
- **Статистика:** отслеживание прогресса обучения
**Организация:** помогает не терять информацию о занятиях, отслеживать их статус и сроки

## Использование

1. **Начало работы:** При первом запуске зарегистрируйте нового пользователя, нажав "Зарегистрируйтесь" в окне входа
2. **Главная панель:** После входа вы попадёте на главное окно, где отображаются ваши недавние события и расписание на сегодня
3. **Основной функционал:** Добавление, редактирование и удаление событий и задач

## Разработка и тестирование

Проект разработан с использованием React.js для создания пользовательского интерфейса.

### Тестирование
В проекте проведено комплексное тестирование, охватывающее основные сценарии использования:

**Тест-кейс** | **Сценарий** | **Охватываемый функционал**
-------------|-------------|----------------------------
Т1 | Полная регистрация и вход | Аутентификация, создание учетной записи
Т2 | Минимальный путь использования | Базовая авторизация, загрузка интерфейса
Т3 | Создание и управление событиями | Добавление, редактирование, удаление учебных событий
Т4 | Работа с задачами | Создание, выполнение, удаление учебных задач
Т5 | Просмотр расписания | Отображение событий на день и неделю
Т6 | Навигация по всем модулям | Переходы между разделами: Расписание, Задачи, Статистика
Т7 | Просмотр статистики | Анализ учебной активности, выполнение задач
Т8 | Поиск и фильтрация событий | Фильтры по дате, типу события
Т9 | Управление профилем | Вход, выход, данные пользователя
Т10 | Работа с календарем | Визуализация событий по датам
Т11 | Тестирование API соединения | Проверка связи с сервером

**Интеграционное тестирование** проводилось на основе графа переходов между окнами приложения, обеспечивая покрытие всех ключевых пользовательских сценариев от регистрации до полного цикла работы с расписанием и задачами.
